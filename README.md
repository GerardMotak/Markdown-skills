# 📓 Markdown Skills — Interactive Notebooks

Welcome! This repo is a hands-on learning resource for anyone who wants to get comfortable writing **Markdown** — from basic text formatting all the way to beautiful mathematical equations.

Whether you're just getting started with Jupyter notebooks or brushing up on LaTeX-style math syntax, you'll find clear examples and practical exercises to work through — no setup required!

No installation needed! Just click any **Open in Colab** badge above and the notebook will open directly in your browser.

## 💡 How to Use the Exercises

Each topic follows the same structure:

- **Quick recap** — a short summary of the syntax
- **Exercise** — a prompt to write your own Markdown
- **Your answer** — a blank cell for you to work in
